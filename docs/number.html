Methods

times(iteratee, timesopt, reverseopt)

Source:
  • Extensions/Function/native.js, line 79
repeats a function n times
Examples

eg. usage

Function.times(5, (i) => {
  console.log(i);
});

// logs 1, 2, 3, 4, 5

or

Function.times(function(i) {
  console.log(i);
}, true);

// logs 5, 4, 3, 2, 1
Parameters:
Name Type Attributes Default Description
iteratee function the iteratee function to invoke
the iteratee will be invoked passing the index as i
so the iteratee has to be something like this
function(i) {}
Properties
Name Type Description
i number the index
times number <optional>
 0 the number of times
reverse boolean <optional>
 false true if you want to do a times reverse cycle
